Caisson lumineux double face bombée

Usage : intérieur / extérieur
Description
Ces caissons sont proposés en version lumineuse ou non-lumineuse.
Ils peuvent être soit horizontaux, soit verticaux.
L'équipement lumineux se fait par tubes fluorescents, cependant un équipement Leds est désormais disponible pour ce produit.
2 modèles sont disponible : verticale ou horizontale.

Possibilté d'équiper le caisson lumineux double face bombée de Leds péripheriques 1,32W
En utilisant la technologie Led, les caissons lumineux double face bombée seront plus économiques.

• Consommation avec équipement Leds peripheriques 1,32 W
Infos produit
Avantages du caisson lumineux double face bombée :

2 fabrications de cette enseigne galbée sont disponibles : Lumineuse avec faces en plexiglas blanc diffusant ou non-lumineuse avec faces tôle laquées blanche.
Les caissons lumineux double face bombée ont 2 possibilités de montage : double face horizontale ou double face verticale.
